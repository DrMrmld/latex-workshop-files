# LaTeX Workshop

Materials for the LaTeX Workshop (19 Nov 2025) organized by the Ministry of Research and Innovations of the Nazarbayev University Student Government and some other files.

Instructors: Madiyar Kassymaly and Azamat Zhaksylykov.

If you want to follow the workshop step by step, go to [this webpage](https://drmrmld.github.io/latex-workshop/).

Descriptions of some directories:

- `example-1-materials`: starting materials for the first example
- `example-1-finished`: the finished source code of example 1 from the workshop
- `example-1-finished-v2`: the finished source code of example 1, but slightly different, because it's the version rewritten for [this webpage](https://drmrmld.github.io/latex-workshop/).
