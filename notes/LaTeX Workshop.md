This is the content of the LaTeX Workshop held on 19 November, 2025, and organized by the Ministry of Research and Innovations of the Nazarbayev University Student Government with Madiyar Kassymaly and Azamat Zhaksylykov as the instructors.

The primary goal of this workshop is to provide you all the foundations, so that you can use LaTeX to write full-fledged documents right away and know how to figure out things if you are stuck.

The workshop consists of the following sections:

- [[Introduction - What is TeX and LaTeX?]]
- [[Example 1 - Creating Simple Documents]]
- [[Example 2 - Making Presentations]]
- [[Getting Help and Further Steps]]

You can find the source code and some other files from this workshop/tutorial in [this GitHub repository](https://github.com/drmrmld/latex-workshop-files).
